About

The Vernacular Cloud project is a multiple-artifact system designed to create feedback loops between a philosophical treatise, a technological taxonomy, and a social media presence. This is version 0.0.1 which is a proof of concept of the semantic web publishing technology. The concept definitions in this version are rudimentary.

The Treatise: The Myth of the Proven

The title of Victor Badinage's masterwork is an obvious riff on Wilfred Sellars' "Myth of the Given." It signals to the academically inclined philosopher that this is a critique of the analytic tradition while remaining accessible to a general audience skeptical of dogmatic certainty.

The Taxonomy: https://vernacular.cloud

The core concepts of the Badinagean nomenclature are published using industry-standard, semantic web technologies. This comprises primarily a SKOS-based content scheme and knowledge graph published both as human-readable HTML pages and as machine readable JSON-LD scripts.
